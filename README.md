201744073 3-C 곽창훈

1)개요
채팅 프로그램 개발

2)목적
소켓, 멀티 스레드를 이용한 채팅 프로그램 개발

3)설계 및 기능
유저들이 중복되지 않고 구분하기 위해 class를 생성하여
ID(이름) 입력
유저의 수를 서버에서 출력
유저가 나갈시 유저수 차감
서버 종료시 채팅프로그램 종료

4)차이점
ID부여
유저의 수 출력
프로그램 종료
